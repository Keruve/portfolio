## Style Guidelines
- **Primary background:** zen-950 (#09090B)
- **Accent colors:** 
  - Orange: #FB923C (3D/creative)
  - Blue: #3B82F6 (engineering)
  - Purple: #A855F7 (highlights)
  - Green: #10B981 (success/availability)
- **Effects:** Glassmorphism, backdrop blur, gradient glows
- **Typography:** Inter (UI), Space Grotesk (headings), JetBrains Mono (code)

## Screenshot Tips
- Use browser DevTools for responsive views
- Enable dark theme
- Capture with realistic content (current projects visible)
- Show interactive states (hover effects, menu open)
- Include favicon in browser tab if visible
- Consider adding subtle shadow/frame for presentation
- Highlight key features: glassmorphic design, animations, project relationships

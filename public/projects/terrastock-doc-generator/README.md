## Style Guidelines
- **Primary colors:** Purple (#A855F7) for automation, Indigo (#6366F1) for templates
- **Dark theme:** zen-950 background (#09090B)
- **Icons:** Document, template, printer, automation, pipeline symbols
- **Visual flow:** Left-to-right transformation (input → process → output)
- **Professional:** Clean, corporate document aesthetics
- **Highlight:** Speed (2+ hours → <5 minutes)
